# 뉴스스탠드 (Newsstand)

코드스쿼드 9-10주차 미션 — 데스크톱 한국형 뉴스 포털.

## 빠르게 시작

```bash
npm install
npm run dev
```

기본 포트: <http://localhost:5173>

## 문서

- `docs/checklist.md` — 기능 단위 체크리스트 (#1-#13, 2주 분할)
- `CLAUDE.md` — AI 페어링 + 디자인 토큰 가이드
- `week9.pdf` — 미션 스펙
- `week9_proposal.pdf` — 6 frame 디자인 캔버스
- `week9_design_system.pdf` — 토큰·컴포넌트·상태 명세

## 스택

- Vite + React 18 + TypeScript
- CSS Variables (디자인 토큰) + CSS Modules
